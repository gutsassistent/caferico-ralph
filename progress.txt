# Ralph Progress Log
## Codebase Patterns
- Exclude the `flowchart` folder from root `tsconfig.json` to keep app typechecking isolated from the React Flow demo.
- Locale switcher pattern: replace the first path segment with the target locale and preserve query params via `usePathname` + `useSearchParams`.
- Scroll reveal pattern: wrap sections/cards in `components/Reveal.tsx` and use the optional `delay` prop for staggered fade-ins.
- Cart state lives in `components/CartProvider.tsx` with `useCart`; the slide-over UI is `components/CartDrawer.tsx` mounted in `app/[locale]/layout.tsx`.
- Parallax accents: use `components/ParallaxOrb.tsx` for scroll-based decorative elements (respects reduced motion).
